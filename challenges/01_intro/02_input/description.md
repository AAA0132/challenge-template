In der "Hello World!"-Challenge hast du ja bereits ein Programm geschrieben, das einen vorgegebenen Text ausgibt. Da es allerdings relativ langweilig ist, wenn das Programm immer genau das gleiche macht, geht es nun darum, zunächst eine Zeile Text einzulesen und diese danach wieder auszugeben. In den eigentlichen Challenges muss dieser Input (der natürlich auch komplexer sein und aus mehreren Zeilen bestehen kann) erst in irgendeiner Weise verarbeitet werden, aber hier geht es zum Reinkommen erstmal nur darum, eine Zeile einzulesen und diese wieder auszugeben.

**Eingabe:**
- Zeile 1: Ein String $s$ bestehend aus $1 \le n \le 100$ Zeichen (Buchstaben, Ziffern, Sonderzeichen)

**Ausgabe:**
- Zeile 1: Der selbe String $s$

***

In the "Hello World!" challenge you have already written a program that outputs a given text. However, since it is relatively boring if the program always does exactly the same thing, the task now is to first read in a line of text and then output it again. In the actual challenges this input (which can of course also be more complex and consist of several lines) must first be processed in some way, but here it is for the time being only about reading in a line and printing it out again.

**Input:**
- Line 1: A string $s$ consisting of $1 \le n \le 100$ characters (letters, digits, special characters)

**Output:**
- Line 1: The same string $s$
